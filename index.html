reactjs教程1

发表于 2016-05-17

这是一系列介绍reactjs＋redux+react-router等开发的教程

  1. 准备

    读到这篇文章的可能是一个萌萌的妹子，她可能还有点天然呆，所以要讲得更细一点。

    首先她要安装以下几个软件:

    git 最新的就可以
    node.js 4.4.4LTS版本

我们要做的是一个单页应用（SPA），所以html页面只有一个就够了

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <link rel="stylesheet" type="text/css" href="./index.css"/>
</head>
<body>
<div id="content"></div>
</body>
<script src="./common.js"></script>
<script src="./index.js"></script>
</html>
  1. 基础react语法

react最常见的写法如下

ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('example')
);

react使用jsx语法，支持html和js混编。
在html中加一对大括号{}就可以在里面写js了。
